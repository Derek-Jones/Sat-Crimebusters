Sea vessel tracking using Satellite AIS data

The Satellite Crimebusters team, at the June 2014 Inventorthon, analysed Automatic Identification System (AIS) data obtained via satellite. The event was hosted by the Satellite Applications Catapult in Harwell.

Team members: Bryan Yap and Derek Jones

Overview

The original purpose of AIS was as a collision avoidance system; knowing where nearby vessels are, their direction and speed, making it easier to avoid colliding with each other. An experiment on the International Space Station showed that it was possible to reliably detect AIS transmissions from space and satellites were launched to obtain world wide information on vessels.

The data provided for the Inventorthon (by exactEarth Ltd) consists of around 250,000 received AIS transmissions per day over 30 days from something like 17,500 distinct vessels travelling around the Mediterranean.

Only vessels over 300 gross tonnage are required to transmit AIS information.

A few vessels appear to be spoofing some fields of the information transmitted, e.g., sending a 4 digit MMSI value when 8 digits are required.

Future patterns to search for

Vessels generally travel between land based locations. Meeting another vessel at sea is an unusual activity. Flagging vessels which appear to be moving slowly in the same location sounds worthwhile.

Distance travelled between transmissions. There is a maximum distance that a vessel can travel between transmissions. Flagging vessels that travel further than some maximum distance distance sounds worthwhile. See R source for details.
